Meta Tags

O que são?

Fornecem informações sobre o conteúdo da página para navegadores e mecanismos de busca

Exemplos:

charset: Define o conjunto de caracteres usados. O padrão atual é 'UTF-8'

viewport: Faz o site se adaptar ao tamanho da tela

description: É a descrição do site, aparece nos resultados de busca do Google

author: Nome de quem criou o site

keywords: Palavras-chave para busca

robots: Diz aos mecanismos de busca se podem indexar seu site ou não

title: Título do documento

description: Entender o conteúdo da página

og:image : Imagem da capa quando compartilha o site

og:url : O link exato do site

og:type : Tipo de conteúdo do site

og:site_name : Nome do site como um todo
